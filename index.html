About Me

My interest in programming began with a desire to build a website to display images I had taken. Throughout the process, I realised that I could learn to build much more powerful programs that solve real world problems and make the lives of users easier and had better. I joined Moringa School a coding bootcamp running for 6 months coding over 1200 hours. There I learned to build full-scale applications over a short time using HTML, CSS and Python and its its frameworks.I also built my resilience and problem-solving skills as I worked to build functional programs. During my time there, I worked in a team of 4 as product owner and software engineer to create an application that allows Speech and Language therapists and their remote assistants to coordinate treatment as well help clients access the services of these professionals. I worked on integrating Django and React which I learnt to build that particular project.
